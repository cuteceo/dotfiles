# dotfiles

Personal dotfiles with cuteDark theme suite for multiple applications.

## Contents

- `themes/` - Theme configurations
  - VS Code (`vscode/cutedark.json`)
  - Zed (`zed/cutedark.json`)
  - Chrome (`chrome/manifest.json`)
  - Zen Browser (`zen/`)
  - Sublime Text (`sublime text/cutedark.sublime-color-scheme`)
  - Discord (`discord/cutedark.css`)
  - Stylus - 130+ website themes (`stylus/`)

## Quick Start

Clone and use whatever you need:
```bash
git clone https://github.com/yourusername/dotfiles.git
```
