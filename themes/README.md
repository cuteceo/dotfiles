# cuteDark Themes

A cohesive dark theme suite for multiple applications, based on the cuteDark color palette.

## Theme List

| App | Path |
|-----|------|
| VS Code | `vscode/cutedark.json` |
| Zed | `zed/cutedark.json` |
| Chrome | `chrome/manifest.json` |
| Zen Browser | `zen/` |
| Sublime Text | `sublime text/cutedark.sublime-color-scheme` |
| Discord | `discord/cutedark.css` |
| Ghostty | `ghostty/` |
| Warp | `warp/` |
| Stylus (Browser Styles) | `stylus/` |

## Stylus Themes

The `stylus/` folder contains userstyles for 130+ websites adapted from Catppuccin to cuteDark.

### Usage

1. Install the [Stylus extension](https://add0n.com/stylus.html)
2. Import `stylus/import.json` in Stylus settings
